# Usage

This is first program in C# and naturally it has to be hello-world

## Compile

On Windows add C:\Windows\Microsoft.NET\Framework\v4.0.30319 to your PATH

```powershell
csc hello-wrold.cs
```

## Run
```powershell
 .\hello-wrold.exe
```
